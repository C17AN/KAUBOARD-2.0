# 항공대 종강시계 2.0

기존의 [항공대 종강시계](https://github.com/Korea-Aerospace-University/KAUBOARD)를 타입스크립트로 재작성하면서, 각종 편의기능 및 소셜 기능을 추가한 새로운 버전입니다.

## 주요 기능

- 종강 / 개강일자 카운트다운
- 실시간 캠퍼스 날씨
- 즐겨찾기 정보 추가
- 메모 / 일정 관리
- 이미지 공유
- 실시간 채팅

## 사용 기술

- `Next.js`
- `Framer motion`
- `Tailwind CSS`
- `Recoil`
- `Firebase Auth + Storage + RTDB`
- `AWS Lambda + API gateway`
